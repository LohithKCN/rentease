# RentEase – Furniture & Appliance Rental Platform

RentEase is a web-based platform that allows students and working professionals to rent furniture and appliances instead of purchasing them.

## Features
- User Registration & Login
- Browse Furniture and Appliances
- Product Image Upload
- Rent Products with Date and Quantity Selection
- Chat Between Renter and Owner
- Admin Dashboard

## Technology Stack
Frontend: HTML, CSS, JavaScript  
Backend: Spring Boot (Java)  
Database: MySQL  

## Database Tables
- Users
- Products
- Rentals
- Messages
